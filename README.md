# Degrees of Confinement

<img width="100%" alt="image" src="https://github.com/user-attachments/assets/40c3207d-467a-492c-9aab-1af0397950db" /><br>

Degrees of Confinement is a browser game that challenges users across ten rounds to distinguish whether a given photo is an american dorm room or a european prison. 

The goal of the project was to explore the overlapping functional requirements of high-density housing and modern rehabilitation centers.

## Play the Demo

[Link to GitHub Pages](https://mediump.github.io/degrees-of-confinement)

## Building Locally

 > If you want to play the game locally on your machine, you will need to [install the Angular CLI through Node.js](https://angular.dev/installation).

After cloning the repository, run `ng serve` to start up the Angular development server.

Alternatively, run `ng build` to compile the project. Build artifacts will be placed in the `dist/` directory.

```bash
# Clone the repo
git clone https://github.com/mediump/degrees-of-confinement.git
cd degrees-of-confinement

# Run the development server
ng serve

# (OR) Build for deployment
ng build
```

This will compile your project and store the build artifacts in the `dist/` directory. By default, the production build optimizes your application for performance and speed.
